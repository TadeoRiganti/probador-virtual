Realidad aumentada para tiendas virtuales

Solucion web para mejorar la experiencia de compra

Sobre el sitio

Es un espacio de experimentacion para jugar con las posibilidades que ofrece el componente web 'model-viewer' (desarrollado por Google), que ofrece tanto renderizado 3D asi como realidad aumentada (ARCore).

Como solución

Se trata de un "mejorador" de experiencias, que busca que el usuario vea el sitio como una herramienta más de compra; y para el cliente del servicio (las mepresas), se busca que mejoren las metricas de los usuarios; que pasen mas tiempo en el sitio, interactue mas, pero finalmente es una solución que busca impactar de manera positiva y memorable.

Propuesta de valor

  1. Para las empresas:
    • Reducir los ciclos de ventas
    • Reducir las tasas de devolucion
    • Ofrecer innovacion como valor de marca
  2. Para los clientes:
    • Brindar más informacion (no verbal) sobre el producto
    • Previsualizar los productos en su espacio (mentalidad funcional)
    • Promover que jueguen con las posibilidades que se ofrecen (mentalidad hedonistica)

Como servicio

Si bien la forma respondera a cada situación en particular, en esencia se trata de:

  • Analizar la estrategia de marca
  • Auditar la experiencia de uso del sitio web
  • Immplementar la funcionalidad (desarrollo directo, indirecto o uso de plugins)
  • Modelado 3D
  • Test de calidad
